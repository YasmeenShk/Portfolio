# 🌐 Personal Portfolio – Yasmeen Shaikh

This is my personal developer portfolio website built to showcase my skills, projects, and contact information.
It highlights my frontend development work, responsive design approach, and UI styling using modern web tools.

---

## 🚀 Live Demo

👉 https://your-live-link-here.com

## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Smooth navigation with fixed blurred header
* Hero section with intro and social links
* About section with developer background
* Projects slider with tech stack icons
* Skills grid with visual icons
* Contact form with EmailJS integration
* Clean footer with branding

---

## 🛠️ Built With

* HTML5
* Tailwind CSS
* JavaScript
* EmailJS (for contact form)
